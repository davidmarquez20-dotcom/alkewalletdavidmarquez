Acerca de este Proyecto

Alke Wallet es una aplicación de billetera digital desarrollada como proyecto de evaluación integradora. El objetivo es proporcionar a los usuarios una solución confiable y fácil de usar para administrar sus activos financieros de manera segura y conveniente.

Características principales: Autenticación segura, gestión de fondos, transferencias validadas, historial de transacciones y contactos.
